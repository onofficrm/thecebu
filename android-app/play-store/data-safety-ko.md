# Play Console — 데이터 보안(Data safety) 작성 참고

Play Console **앱 콘텐츠 → 데이터 보안** 설문 시 아래를 참고하세요.  
실제 수집·이용 방식이 바뀌면 이 문서와 스토어 항목을 함께 수정하세요.

---

## 수집 여부

**예** — 앱이 사용자 데이터를 수집·공유합니다.

(계정·위치·UGC가 있으므로 "수집 안 함"은 선택 불가)

---

## 수집하는 데이터 유형 (체크 후 세부 입력)

| 데이터 | 수집 | 공유 | 필수/선택 | 목적 |
|--------|------|------|-----------|------|
| 이름 / 닉네임 | 예 | 아니오 | 필수(가입) | 계정 관리 |
| 이메일 주소 | 예 | 아니오 | 필수(가입) | 계정 관리, 고객 지원 |
| 전화번호 | 선택적 | 아니오 | 선택 | 계정·연락 (수집 시) |
| 대략적 위치 | 예 | 아니오 | 선택(GPS 권한) | 앱 기능(내주변) |
| 사진 / 동영상 | 선택적 | 아니오 | 선택 | 사용자 콘텐츠(게시글 첨부) |
| 기타 사용자 생성 콘텐츠 | 예 | 아니오 | 선택 | 커뮤니티·톡방 |
| 앱 상호작용 | 선택적 | 아니오 | — | 분석(사용 시에만) |
| 기기 ID | 아니오* | — | — | — |

\* Firebase/Analytics를 나중에 붙이면 이 항목을 다시 검토하세요.

---

## 데이터 처리 방식

| 질문 | 답변 |
|------|------|
| 데이터가 암호화되어 전송되나요? | **예** (HTTPS) |
| 사용자가 데이터 삭제를 요청할 수 있나요? | **예** — MY 탈퇴 또는 help@thecebu.co.kr |
| 데이터 수집이 필수인가요? | 위치·UGC는 **선택**, 계정 정보는 **필수** |

## 심사 시 유의

- **위치**: "내주변" 기능 설명과 일치해야 함
- **UGC**: 신고·차단·운영 정책이 사이트에 있으면 콘텐츠 등급 설문에 반영
- **소셜 로그인**(Google 등): 해당 제공자로부터 받는 항목도 데이터 보안에 포함
